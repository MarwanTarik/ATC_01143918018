# Prerequisites
- [.NET 9 SDK]
- [React v18]

# Setup
### 1. Clone the Repository
```bash
git clone https://github.com/MarwanTarik/ATC_01143918018.git
cd ATC_01143918018
```

### 2. Restore Dependencies
dotnet restore

### 3. Build the Project
dotnet build

### 4. Run the API
dotnet run

The API will launch at:
http://localhost:5105

### 5. Documentation
Navigate to http://localhost:5105/scalar in development mode.
