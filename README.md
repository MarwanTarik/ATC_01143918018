# Prerequisites
- [.NET 9 SDK]
- [React v18]

# Setup
### 1. Clone the Repository
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

### 2. Restore Dependencies
dotnet restore

### 3. Build the Project
dotnet build

### 4. Run the API
dotnet run

The API will launch at:
http://localhost:5105

### 5. Documentation
Navigate to http://localhost:5105/scalar in development mode.
